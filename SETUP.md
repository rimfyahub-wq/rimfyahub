# Glow Edge Shopify Theme - Setup Instructions

## After Uploading the Theme:

1. **Go to Theme Customizer**
   - Admin → Online Store → Themes
   - Click "Customize" on the Glow Edge theme

2. **Add Sections to Homepage**
   - Click "Add section"
   - Add "Hero" section at the top
   - Add "Features" section below Hero
   
3. **Configure the Hero Section**
   - Click on the Hero section
   - The default content should appear automatically
   - Optionally upload a video or image

4. **Configure Features Section**
   - Click on the Features section  
   - You should see 6 feature blocks already added
   - Each has icon, title, description, and glow color pre-configured
   - If blocks are missing, click "Add block" and select "Feature"

5. **Save and Publish**
   - Click "Save" in the top right
   - The theme should now display with all effects active

## Troubleshooting:

### No Components Showing:
- Make sure you've added the Hero and Features sections in the customizer
- Check that blocks are added to the Features section
- Try refreshing the customizer preview

### JavaScript Not Working:
- Open browser console (F12)
- Look for "Theme loaded successfully" message
- Check for any JavaScript errors

### Missing Glows/Effects:
- Clear browser cache
- Make sure JavaScript is enabled
- Check that the theme.css and JavaScript files are loading
